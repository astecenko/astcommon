В работе применяю следующие бибилиотеки и компоненты:
1. FastMM - бесплатный улучшенный менеджер памяти (с 2009 Delphi идет уже с самой Delphi)
2. FastCode - бесплатный набор модулей с оптимизированными и исправленными стандартными функциями
3. FastMove - -"-
4. VCLFixPack - -"- 
5. NativeExcel - объекты для вывода в формат Ms Excel без необходимости наличия устновленного Ms Excel (массивы в памяти, если большие таблицы то при выводе может занять всю ОЗУ) 
6. EhLib 5.2 - улучшенные таблицы
7. JEDI VCL (JVCL) - бесплатный набор компонентов
8. madCollection (madExcept) - профайлер
9. VK DBF - набор компонентов для работы с таблицами в формате DBF
10. Модули из репозитория  http://code.google.com/p/astec-docs/
  - ArrayS.pas - реализация ассоциативных массивов в Delphi
  - CommandLine.pas - объект для оперирования параметрами командной строки программы
  - MsAD.pas - набор функций для работы с Ms Active Directory
  - SAVLIB[_DB,_DBF,_HTML,_INI].pas - наборы различный функций
  - SevenZip.pas - API для работы с архивами посредством 7z.dll 
  - ShellLink.pas - функция для создания ярлыков
  - SingletonTemplate.pas - реализация паттерна Singleton в Delphi
  - SPGetSid.pas - функции для получения и обработки идентификаторов безопасности (SID) в OC Ms Windows 
  - Support.pas - функции для получения снимков экрана и работы с графическими файлами
